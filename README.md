# Earning Fish

Earning Fish is a Flutter-based PDI (Pre-Delivery Inspection) management application focused on secure, end-to-end encrypted data communication, real-time updates, and PDF report generation. The app provides a smooth and modern UI/UX with animations and custom widgets while ensuring data security and modular clean architecture for maintainable code.

# Why? What?
Earning Fish focuses on secure and efficient PDI generation for automotive operations. By combining real-time updates, end-to-end encryption, and easy report generation, it ensures teams can track, manage, and share inspections securely and efficiently.

# Project Overview
Earning Fish is designed to empower automotive teams by providing a secure and efficient platform for managing Pre-Delivery Inspections (PDI). Users can log in safely via Firebase Authentication and access a dashboard that displays PDI statuses, including total, pending, and completed inspections. The app allows teams to create and manage detailed PDI forms for each vehicle, generate PDF reports of completed inspections, and share them easily through WhatsApp, Email, or other channels. With end-to-end encrypted communication and secure storage using Flutter Secure Storage and Firebase Cloud Storage, all data remains safe and protected. Additionally, Earning Fish supports live data updates, enabling real-time tracking of PDI statuses and providing teams with an up-to-date overview on the dashboard.

# Key Features
- Splash Screen with Lottie Animations for a polished onboarding experience.
- Firebase Authentication for secure login.
- Custom UI/UX with clean and modular design.
- Dashboard with filters for Pending, Completed, and Total PDIs.
- PDI Form Management with dynamic fields and data validation.
- PDF Generation for PDI reports.
- Report Sharing via WhatsApp, Email, etc.
- End-to-End Data Encryption for secure storage and communication.
- Real-Time Updates with Bloc/Cubit state management.
- Clean Architecture with modular codebase for maintainability.

# Tech Stack & Tools
Frontend: Flutter & Dart | State Management: Bloc & Cubit | Backend / Cloud: Firebase | Animations: Lottie Animations | PDF Generation | Architecture: Clean Architecture (SOLID principles, modular code) | Version Control
